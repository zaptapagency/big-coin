Run the desktop / instant miner

  1. Install Python 3 (tick “Add to PATH” on Windows).
  2. Download moonbite-miner.py and your OS launcher into the same folder.
  3. Get a MoonBite address (moon1… or M…) from your wallet.
  4. Run: python moonbite-miner.py --address YOUR_ADDRESS — or just double-click the launcher.
  5. Watch blocks land. Press Ctrl+C to stop.
Full node + GUI

MoonBite Core wallet

The real desktop wallet — a full node with a graphical send/receive/mine interface. Runs the true MoonBite Core (v0.21.5.5), generates moon1… addresses, and syncs the chain itself.

Download wallet — Linux x86-64 (.tar.gz)

Bundle: moonbite-qt · moonbited · moonbite-cli · sample config · SHA256SUMS — ~12 MB

  1. tar -xzf moonbite-wallet-linux-x86_64.tar.gz && cd moonbite-wallet
  2. chmod +x moonbite-qt moonbited moonbite-cli
  3. ./moonbite-qt — the wallet creates ~/.moonbite on first run.

Linux x86-64 only. On Windows, run it inside WSL2 (Ubuntu 22.04), or use the browser / one-click miner above. See README.txt in the bundle for dependencies and headless-node steps.

Honest note: MoonBite is pre-mainnet — no coins have market value yet. Mining is enabled by the explorer operator (MINING_ENABLED). If a miner says “mining endpoint is disabled”, it simply hasn’t been switched on for that chain.
