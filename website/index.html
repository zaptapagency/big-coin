Built on proven technology

BigCoin doesn't reinvent the wheel. It inherits a decade of security engineering from Bitcoin Core and Litecoin, with parameters tuned for accessible mining.

Truly Mineable

Scrypt proof-of-work is memory-hard and CPU/GPU friendly, so ordinary hardware can take part in securing the network from day one.

Fast Confirmations

A 2.5-minute target block time means transactions settle roughly four times faster than a 10-minute chain, keeping the ledger responsive.

Open Source

Forked from the battle-tested Litecoin and Bitcoin Core codebases. The full source is auditable — read it, build it, verify it yourself.

Fair Launch

A premine of 1% or less means coins are distributed primarily through open mining rather than reserved for insiders.

Secure by Design

Cumulative proof-of-work plus a UTXO transaction model gives the network the same well-understood security foundations as Bitcoin.

Your Keys, Your Coins

You hold your own private keys in a non-custodial wallet. No account, no gatekeeper, no third party that can freeze your funds.

How to mine BigCoin

Three steps to point hash power at the network. No permission required.

1

Download the wallet / node

Install BigCoin Core (Qt) or run the headless daemon to hold a copy of the chain and receive your rewards.

2

Get a Scrypt miner or join a pool

Use a Scrypt-capable miner such as cpuminer or a GPU miner. Solo mine, or join a pool to smooth out payouts.

3

Point it at the network

Aim your miner at your local node's RPC or a pool's stratum endpoint using a BigCoin address (starts with “B”).

terminal — solo mine against a local node 
# 1. Start the BigCoin daemon and let it sync
bigcoind -daemon -server -rpcuser=big -rpcpassword=changeme

# 2. Create a receiving address for your rewards
bigcoin-cli getnewaddress "mining"
B1qMinerAddrExampleDoNotUse00000000

# 3. Point a Scrypt miner at the node (example: cpuminer)
minerd --algo scrypt \
  --url http://127.0.0.1:9332 \
  --user big --pass changeme \
  --coinbase-addr B1qMinerAddrExampleDoNotUse00000000

Commands are illustrative. Always verify software signatures and use a password you generate yourself before running anything.

Roadmap

A deliberate, security-first path. Steps ship when they're ready — not before.

  1. In progress

    Testnet

    A public test network for validating consensus rules, difficulty retargeting, and wallet behaviour with valueless coins.

  2. Planned

    Security Audit

    Independent review of consensus changes and packaging before any value is at stake on mainnet.

  3. Planned

    Mainnet Launch

    Genesis block and public release of BigCoin Core, opening fair mining to everyone.

  4. Planned

    Block Explorer

    A public explorer to inspect blocks, transactions, and addresses in real time.

  5. Planned

    Exchange Listings

    Community-driven liquidity starting with decentralized exchanges (DEX), followed by centralized exchanges (CEX) as adoption grows.
