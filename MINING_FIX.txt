                         STEP-BY-STEP FIX
================================================================================

1. KILL ALL SERVERS
   - Close all 3 terminals (or Ctrl+C each one)
   - Wait 5 seconds to make sure they're stopped

2. START FRESH (in this order)

   Terminal 1:
   -----------
   cd C:\Users\usman\Desktop\BigCoinBB
   python shared_state.py

   Wait for output: "[Shared State Server] Listening on 127.0.0.1:9999"

   Terminal 2:
   -----------
   cd C:\Users\usman\Desktop\BigCoinBB
   python web_app_shared.py

   Wait for output: "Running on http://127.0.0.1:5000"

3. TEST IN BROWSER
   - Open: http://localhost:5000
   - Go to Wallet tab

4. TRY MINING
   - Go to Mining tab
   - Paste address in "Miner Address" field
   - Set "Blocks" to 3
   - Click "Start Mining"

EXPECTED RESULT:
  - Progress bar appears
  - Updates every 500ms (1/3, 2/3, 3/3)
  - No errors
  - No freezing
  - Mining completes in 10-60 seconds
