SETUP INSTRUCTIONS
================================================================================

1. INSTALL FLUTTER (if not already installed):
   Download from: https://flutter.dev/docs/get-started/install
   Verify: flutter --version && flutter doctor

2. NAVIGATE TO PROJECT:
   cd C:/Users/usman/Desktop/BigCoinBB/mobile

3. INSTALL DEPENDENCIES:
   flutter pub get

4. GENERATE JSON MODELS (must do once after creating/modifying models):
   flutter pub run build_runner build

5. RUN ON ANDROID:
   - Start Android emulator or connect device
   - flutter run

6. RUN ON iOS:
   - Start iOS simulator (Xcode required on macOS) or connect device
   - flutter run

7. BUILD FOR RELEASE:
   Android: flutter build apk --release
   iOS: flutter build ios --release
