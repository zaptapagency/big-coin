================================================================================
                       NETWORK PROTOCOL
================================================================================

Handshake (initial connection):
  Node A → Node B: {type: "handshake", node_id: "alice", ...}
  Node B → Node A: {type: "handshake", node_id: "bob", ...}
  ✓ Connection established

Block Broadcasting (mining):
  Node A mines block → {type: "new_block", block: {...}}
  Broadcasts to all peers
  All peers receive and validate
  ✓ All blockchains synchronized

Keep-Alive (30 second interval):
  Node A → Node B: {type: "ping"}
  Node B → Node A: {type: "pong"}
  ✓ Peer still alive

Synchronization (10 second interval):
  Node A → Peers: {type: "sync_request"}
  If peer height > my height:
    Request blocks from (my_height+1) to peer_height
  ✓ Blockchain synchronized
