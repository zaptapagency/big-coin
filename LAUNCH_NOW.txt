================================================================================
                         TROUBLESHOOTING
================================================================================

Problem: "Address already in use"
Solution:
  taskkill /F /IM python.exe
  Wait 5 seconds
  Run: LAUNCH_EVERYTHING.bat


Problem: Nodes won't connect
Solution:
  Make sure all 3 nodes are running
  Check: ps aux | grep python (should show 3)
  Run: python connect_nodes.py


Problem: Web dashboard won't load
Solution:
  Open: http://localhost:8001
  If doesn't work:
    - Check port 8001 is open
    - Make sure Alice node is running
    - Try refreshing browser


Problem: Mining is slow
Solution:
  That's NORMAL! Mining takes 10-60 seconds
  Just wait patiently
  Check other nodes (8002, 8003) - height should increase


Problem: Height doesn't match between nodes
Solution:
  Wait 30 seconds (background sync runs every 10s)
  Run: python connect_nodes.py
  Check if nodes are connected (should show Connected Peers: 2)


For more help:
  Read: P2P_SETUP_GUIDE.md (Troubleshooting section)
